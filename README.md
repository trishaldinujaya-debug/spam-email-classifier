# Spam Email Classifier

This project implements spam email classification using the Spambase dataset from the UCI Machine Learning Repository.

## Algorithms Used

* K-Nearest Neighbours (KNN)
* Naive Bayes

## Tools

* Python
* Google Colab
* Pandas
* Scikit-learn

## Dataset

Spambase Dataset (UCI Machine Learning Repository)

## Objective

To classify emails as spam or non-spam using machine learning techniques.
